**注意事项：**
1. 所有图标建议使用纯色或简单渐变，保持视觉一致性
2. 主题色为 #1989fa（蓝色），次要色为 #969799（灰色）
3. 选中状态图标使用主题色，未选中使用灰色
4. 建议导出 @2x 和 @3x 版本以适配不同屏幕
5. 可以使用 iconfont.cn 或 icons8.com 等图标库获取图标
